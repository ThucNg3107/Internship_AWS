This article shares detailed insights on how Amazon's "Your Orders" development team improved customer experience by introducing semantic search capabilities into their existing lexical search system, utilizing Amazon OpenSearch Service and Amazon SageMaker, overcoming large-scale scalability challenges and maintaining absolute system availability.

###  [Blog 2 - Simplifying AWS AppSync Events Integration Using Powertools for AWS Lambda](3.2-Blog2/)
This article introduces how to use the AppSyncEventsResolver in the Powertools for AWS Lambda toolkit to handle real-time events from AWS AppSync. ###  [Blog 3 - Amazon S3 Audit Logging](3.3-Blog3/)
This article is Part 2 in a series about audit logging on Amazon S3, focusing on identity-based security investigations using AWS CloudTrail data events and Amazon Athena. You will learn how to set up organizational logging, optimize Athena tables with partition projection for cost savings, and apply common SQL query templates to monitor user activity, cross-account access, and failed access attempts.
