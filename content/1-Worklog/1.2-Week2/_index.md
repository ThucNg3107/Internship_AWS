### Week 2 Objectives:

* Manage EC2 instances (Linux & Windows), manage EBS volumes/snapshots, AMIs, and permissions via IAM Roles for EC2.
* Set up remote development environment using Cloud9, host static websites on S3, and configure basic Amazon RDS databases.
* Learn containerization using Docker with ECR and ECS, and set up monitoring with CloudWatch, Container Insights, Security Hub, and AWS Backup.
* Build advanced networking (VPC Peering, Transit Gateway) and establish CI/CD pipelines via GitLab, GitHub Actions, and AWS CodeBuild.
