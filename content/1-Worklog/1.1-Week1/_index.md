### Week 1 Achievements:

* Understood what AWS is and gained a solid grasp of the core service groups:
  * Compute
  * Storage
  * Networking
  * Database
  * ...

* Successfully created and configured an AWS Free Tier account.

* Became familiar with AWS Management Console and learned how to find, access, and use services through the web interface.

* AWS onboarding and collaboration included:
  * Got acquainted with FCJ team members, understood internship regulations, and learned core AWS cloud service concepts (Compute, Storage, Networking, Database).

* Identity and Access Management (IAM):
  * Proficient in creating and managing IAM Users, Groups, Roles, and Policies.
  * Gained deeper understanding of advanced security mechanisms such as role switching (Assume Role), policy types, and applying IAM Instance Profile for EC2 instances.

* Acquired the ability to use both web console and CLI in parallel to manage AWS resources.

* Cloud Networking (VPC):
  * Independently designed and deployed a complete Virtual Private Cloud (VPC) architecture.
  * Configured core networking components in detail, including Subnet, Route Table, Internet Gateway, NAT Gateway, Security Group, VPC Flow Logs, Elastic IP, and learned Site-to-Site VPN connectivity.
