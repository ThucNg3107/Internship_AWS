### Week 1 Achievements:

* **Onboarding & Collaboration:**
  * Completed integration into FCJ team culture and reviewed internship rules with team members.

* **AWS Foundation & Cost Management:**
  * Successfully created and configured an AWS Free Tier account, getting familiar with the AWS Management Console.
  * Deeply understood and practiced configuring AWS Budgets to set up reasonable cost alerts.
  * Accessed AWS Support to learn how to submit technical support cases based on severity levels.

* **Security & Access Management (IAM):**
  * Mastered the AWS IAM service through configuring Groups, Users, Roles, and Policies, and applying the Assume Role mechanism.
  * Deeply understood security mechanisms such as Assume Role, distinguishing between Policy types (Identity-based, Resource-based), and applying IAM Instance Profiles for EC2 virtual servers.
  * Successfully installed and configured the AWS CLI on a personal computer for programmatic interaction with AWS resources.

* **Cloud Network Configuration (VPC):**
  * Independently designed and successfully deployed a complete Virtual Private Cloud (VPC) following the AWS Well-Architected Framework.
  * Configured core network components in detail, including Subnets (Public/Private), Route Tables, Internet Gateways, NAT Gateways, and Security Groups.
  * Used EC2 Instance Connect Endpoint to securely connect to virtual servers in private networks.
  * Explored theories of Site-to-Site VPN connections and monitored network traffic using VPC Flow Logs.
