### Week 3 Objectives:

* Deploy automation and CI/CD with AWS Lambda, EventBridge, CodePipeline, and CodeDeploy.
* Manage hybrid storage with AWS Storage Gateway, application security with AWS WAF, and resource management using Tags and Resource Groups.
* Configure advanced monitoring with Grafana/CloudWatch, establish permissions via IAM Permission Boundaries, and patch management with Systems Manager.
* Practice data security with AWS KMS, CloudTrail, set up a Data Lake, and use AWS Glue and Athena for data querying.

### Week 3 Achievements:

* **Serverless Computing & Deployment Automation:**
  * Configured Lambda functions and EventBridge.
  * Deployed applications on EC2 using AWS CodePipeline and CodeDeploy.

* **Security, Storage & Resource Management:**
  * Configured AWS Storage Gateway for hybrid data synchronization.
  * Implemented AWS WAF rules to protect web applications.
  * Efficiently organized resources using tags and AWS Resource Groups.
  * Configured IAM Permission Boundaries and Organization Policies.
  * Practiced data encryption (encryption at rest) with AWS KMS and monitored activities using CloudTrail.

* **Monitoring & System Operations:**
  * Integrated Grafana with CloudWatch for robust resource monitoring.
  * Managed OS patching across multiple servers using AWS Systems Manager (Run Command).

* **Data Engineering & Infrastructure as Code (IaC):**
  * Built a basic Data Lake architecture using AWS CDK.
  * Used AWS Glue and Athena to query, transform data, and analyze performance.
  * Safely shared encrypted data over Amazon S3.
